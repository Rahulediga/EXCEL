# EXCEL - EcoWaste IoT Dashboard

EcoWaste IoT system dashboard for monitoring and managing waste efficiently.

## Features
- Real-time bin status monitoring
- AI-driven insights and analytics
- Alert system for waste management
- Comprehensive reporting

## Tech Stack
- **Frontend**: React + Vite
- **Backend**: Python (FastAPI)
- **Database**: Firebase / Firestore

## Setup
1. `npm install`
2. `npm run dev`
